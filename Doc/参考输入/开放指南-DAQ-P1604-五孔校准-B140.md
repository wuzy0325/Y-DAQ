### 3.3 通道结构

18 通道：CH1~CH16（压力）+ CH17（大气压）+ CH18（大气温度）

```
通道索引:  0    1    2   ...  15   16      17
通道名称: CH1  CH2  CH3  ... CH16  大气压  大气温度
```

### 3.4 通信协议

**帧格式**: `| lenHi(1B) | lenLo(1B) | payload...(len-2 字节) |`

2 字节大端长度前缀，前 2 字节为长度本身，后续为有效载荷。

**命令列表**:

| 命令 | 功能 | 响应 |
|------|------|------|
| `w1601` | 启用 2 字节长度前缀模式 | ASCII 确认 |
| `c 00 <st> FFFF 1 <per> 7 0` | 配置数据流参数（内部时钟/大端/连续） | ASCII 确认 |
| `c 05 <st> 0810` | 配置数据流返回内容（压力+大气压+温度） | ASCII 确认 |
| `c 01 <st>` | 启动数据流 | 二进制数据帧 |
| `c 02 <st>` | 停止数据流 | ASCII 确认 |
| `rFFFF0` | 轮询 16 通道压力（备用调试） | ASCII 数据 |

**数据流帧结构**:
```
| 头(5B) | CH1(4B, float32 BE) | CH2 | ... | CH16 | CH17(大气压) | CH18(温度) |
```
共 5 + 18×4 = 77 字节

### 3.5 连接流程

```
connect()
  ├─ 创建 TCP Socket (net.Socket)
  ├─ 连接到 host:port
  ├─ 发送 w1601 启用长度前缀模式
  ├─ 注册 Socket 事件:
  │   ├─ 'data' → handleSocketData() — 2字节长度前缀拆包
  │   ├─ 'error' → handleDisconnect() — 触发自动重连
  │   └─ 'close' → handleDisconnect()
  └─ 返回 true
```


### 3.6 采集流程

```
startAcquisition()
  ├─ configureStream(periodMs)
  │   ├─ c 00: 内部时钟 / 大端二进制 / 连续采集
  │   └─ c 05: 返回压力+大气压+温度
  ├─ 发送 c 01 启动数据流
  └─ 返回 true

数据接收:
  handleSocketData(chunk)
    ├─ 追加到 recvBuffer
    ├─ 按 2 字节长度前缀拆包
    ├─ 判断帧类型 (ASCII / 二进制)
    ├─ 二进制帧 → handleStreamFrame()
    │   ├─ 解析 5 字节头 + 18 个 float32(BE)
    │   ├─ 映射到通道数据
    │   └─ 调用 onDataCallback(DataPayload)
    └─ ASCII 帧 → 命令响应队列
```

**源码关键片段** (`src/main/drivers/DAQP1604Device.ts:402-450`):

```typescript
private handleStreamFrame(frame: Buffer): void {
  // 解析 5 字节头 + 18 个 float32(BE)
  const values: number[] = []
  for (let i = 0; i < 18; i++) {
    const offset = 5 + i * 4
    values.push(frame.readFloatBE(offset))
  }
  // 映射到已启用通道，构造 DataPayload
  // 调用 this.onDataCallback(payload)
}
```

### 4.8 编码器补偿状态机

B140 实现了完整的编码器补偿，确保运动到位精度：

```
waiting-stop → settling → checking → compensating → succeeded / failed / cancelled
```

**补偿流程**:
1. `moveTo`/`moveBy` 执行后，若轴启用了编码器补偿，创建 `PendingCompensationRequest`
2. 在 `getAllAxisStatus` 轮询时，检测到运动停止后激活补偿任务
3. 补偿循环:
   - 读取编码器实际位置
   - 计算与目标位置的误差
   - 若误差在容差内 → 同步寄存器并标记成功
   - 否则 → 发送修正运动并继续等待
4. 支持最大循环次数、超时、最小步长等保护参数

**补偿配置** (AxisEncoderCompensationConfig):

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `enabled` | `false` | 是否启用 |
| `tolerance` | `0.01` | 补偿容差（工程单位） |
| `maxCycles` | `3` | 最大补偿次数 |
| `settleMs` | `100` | 稳定等待时间(ms) |
| `minStep` | `0` | 最小补偿步长 |
| `timeoutMs` | `5000` | 补偿超时(ms) |

### 4.9 串行命令锁

B140Driver 使用 `runExclusive(fn)` 保证同一时刻仅一条命令在传输，避免命令交错。

**源码关键片段** (`src/main/drivers/B140MotionController.ts:333-340`):

```typescript
private async runExclusive<T>(fn: () => Promise<T>): Promise<T> {
  const next = this.commandLock.then(fn)
  this.commandLock = next.then(() => undefined, () => undefined)
  return await next
}
```

### 4.10 状态轮询

- 频率: 10Hz (100ms 间隔)
- 每次轮询遍历所有已连接实例，调用 `driver.getAllAxisStatus()` 获取最新状态
- 通过 `BrowserWindow.webContents.send('motion:status-updated', ...)` 广播到前端
- 含防重入队列机制和限位/补偿错误检测

```typescript
  for (let i = 0; i < this.config.samplesPerPoint; i++) {
    const snapshot = await this.getChannelBatch(enabledChannels, options)
    const sample = this.readRawData(snapshot.values)  // 通道匹配
    samples.push(sample)
    
    // 实时推送（100ms 节流）
    const coefficients = calculateFiveHoleCoefficients(sample)
    this.sendRealtimeUpdate({ point, fiveHoleRaw: sample, fiveHoleCoefficients: coefficients })

  // 3. 计算系数

  // 4. 计算标准差
```

### 5.5 校准系数计算公式

**文件**: `src/main/calibration/formulas.ts`

```typescript
// 五孔探针系数计算
function calculateFiveHoleCoefficients(data: FiveHoleRawData): FiveHoleCoefficients {
  const { p1, p2, p3, p4, p5, pAtm, pTotal } = data

  // 平均压力
  const pAvg = (p2 + p3 + p4 + p5) / 4

  // 分母（避免除零）
  const denominator = p1 - pAvg
  const safeDenominator = Math.abs(denominator) < 1e-6 ? 1e-6 : denominator

  // 攻角系数 Kα
  const Kalpha = (p2 - p3) / safeDenominator

  // 侧滑角系数 Kβ
  const Kbeta = (p4 - p5) / safeDenominator

  // 总压系数 CPT
  let CPT = 0
  if (pTotal !== undefined && pTotal > pAtm) {
    CPT = (p1 - pAtm) / (pTotal - pAtm)
  }

  // 静压系数 CPS
  let CPS = 0
  if (p1 > pAtm) {
    CPS = (pAvg - pAtm) / (p1 - pAtm)
  }

  return { Kalpha, Kbeta, CPT, CPS }
}
```

**公式汇总**:

| 系数 | 公式 | 说明 |
|------|------|------|
| Pavg | (P2 + P3 + P4 + P5) / 4 | 四孔平均压力 |
| Kα | (P2 - P3) / (P1 - Pavg) | 攻角系数 |
| Kβ | (P4 - P5) / (P1 - Pavg) | 侧滑角系数 |
| CPT | (P1 - P∞) / (Pt - P∞) | 总压系数（需总压） |
| CPS | (Pavg - P∞) / (P1 - P∞) | 静压系数 |
