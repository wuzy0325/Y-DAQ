# 三孔探针插值算法业务逻辑说明

## 1. 概述

三孔探针插值算法的作用是：根据三孔探针三个压力孔的实测压力值，结合预先标定的校准数据，反算出气流的**总压**、**静压**、**马赫数**和**攻角**。

核心思路：先利用三个孔的压差计算出无量纲压力系数 Kb，然后在校准数据中进行二维插值（马赫数方向 + 攻角方向），通过迭代逐步逼近真实的马赫数和攻角。

---

## 2. 输入

### 2.1 测量数据

每条测量记录包含以下字段：

| 字段 | 含义 | 单位 | 是否参与算法 |
|------|------|------|-------------|
| Pa | 大气压 | Pa | 是 |
| Ta | 大气温度 | °C | 否（仅存储） |
| PtInlet | 进口总压 | Pa | 否（仅存储） |
| PsInlet | 进口静压 | Pa | 否（仅存储） |
| ProbeHole1 | 1号探针孔压力 | Pa | 是 |
| ProbeHole2 | 2号探针孔压力（中心孔） | Pa | 是 |
| ProbeHole3 | 3号探针孔压力 | Pa | 是 |

**算法实际使用的输入量**：大气压 Pa、三个探针孔压力 P1/P2/P3。

### 2.2 校准数据

可以加载一个或多个校准文件，每个文件对应一个校准马赫数 CMa，包含若干组攻角下的标定系数。

每个校准文件的结构：

```
第1行：CMa              （校准马赫数，如 0.2、0.4、0.6）
第2行：Nalpha           （攻角数据条数，如 13）
第3~Nalpha+2行：Kb  Kt  Sb  Alpha   （每行4列）
```

**校准系数含义**：

| 系数 | 含义 | 物理意义 |
|------|------|---------|
| Kb | 压力系数 | 反映两侧孔压差与总压差之比，用于反查攻角 |
| Kt | 总压系数 | 用于从中心孔压力修正得到来流总压 |
| Sb | 静压系数 | 用于从总压修正得到来流静压 |
| Alpha | 攻角 | 校准时的气流攻角，单位：度 |

**示例**（校准马赫数 Ma = 0.4，攻角范围 -30° ~ +30°，步长 5°）：

```

  ...（共13行）

```

## 4. 算法总体流程

### 4.1 预处理

加载所有校准文件后，提取以下全局参数：

- **攻角序列** alphaOri：所有校准文件共用的攻角列表（如 -30, -25, -20, ..., +30）
- **初始马赫数** initMa：所有校准马赫数的算术平均值
- **马赫数下限** minMa：所有校准马赫数中的最小值
- **马赫数上限** maxMa：所有校准马赫数中的最大值

### 4.2 逐条处理

对测量数据中的每一行，执行以下三个步骤：

```
步骤1：计算测量的压力系数比 Kb_temp
步骤2：迭代求解马赫数和攻角（最多20次）
步骤3：用最终结果计算输出值
```

---

## 5. 步骤1：计算测量的压力系数比

**目的**：从三个探针孔的原始压力值计算出一个无量纲的压力系数比，作为后续反查攻角的依据。

**计算公式**：

先计算压力差分：

$$
\Delta P = 2 \times P_2 - P_1 - P_3
$$

再计算压力系数比：

$$
Kb_{temp} = \frac{P_3 - P_1}{\Delta P} = \frac{P_3 - P_1}{2P_2 - P_1 - P_3}
$$

其中 P1、P2、P3 分别是1号孔、2号孔（中心孔）、3号孔的压力值。

**异常处理**：
- 若 ΔP 的绝对值小于 $10^{-6}$（分母接近零），无法计算，直接输出 Ma=0、Alpha=0、Pt=P2、Ps=P2
- 若 Kb_temp 为无穷大或非数值，直接输出 Ma=initMa、Alpha=0、Pt=P2、Ps=P2

---

## 6. 步骤2：迭代求解马赫数和攻角

### 6.1 迭代参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始马赫数 | 所有校准 CMa 的平均值 | 迭代起点 |
| 最大迭代次数 | 20 | 防止死循环 |
| 收敛容差 | $10^{-4}$ | 马赫数变化量小于此值则认为收敛 |

### 6.2 单次迭代流程

每次迭代依次执行以下五个子步骤：

**子步骤 A：在当前马赫数下查找匹配的攻角和系数**

根据当前马赫数和测量的 Kb_temp，通过二维插值在校准数据中查找对应的攻角 Alpha 和系数 Kt、Sb。（详见第7节）

**子步骤 B：计算总压和静压**

$$
Pt = P_2 + Kt \times \Delta P
$$

$$
Ps = Pt - Sb \times \Delta P
$$

其中 Kt 和 Sb 由上一步插值得到。Pt 和 Ps 均为表压（相对大气压）。

**子步骤 C：计算新的马赫数**

$$
Ma_{new} = \sqrt{5 \times \left| \left( \frac{Pt + Pa}{Ps + Pa} \right)^{0.2857} - 1 \right|}
$$

这是等熵流动关系式的反算，详见第8节。

**子步骤 D：收敛判断**

若 $|Ma_{new} - Ma_{current}| < 10^{-4}$，则认为已收敛，退出迭代。

**子步骤 E：更新马赫数**

将新马赫数限制在校准范围内，作为下一次迭代的输入：

$$
Ma_{current} = \max(minMa, \; \min(maxMa, \; Ma_{new}))
$$

### 6.3 迭代伪代码

```
Ma_current ← initMa

重复（最多20次）：
    （Alpha, Kt, Sb）← 二维插值查找（Kb_temp, Ma_current）
    Pt ← P2 + Kt × ΔP
    Ps ← Pt − Sb × ΔP
    Ma_new ← 等熵公式反算（Pt, Ps, Pa）
    Ma_new ← 限制在 [minMa, maxMa] 范围内

    若 |Ma_new − Ma_current| < 0.0001：
        Ma_current ← Ma_new
        跳出循环

    Ma_current ← Ma_new
```

## 7. 核心二维插值算法详解

这是整个算法的核心部分：在给定马赫数下，根据测量的 Kb 值在校准数据中反查出攻角和系数。

整个插值分为两个方向：**马赫数方向**（在校准马赫数之间插值）和**攻角方向**（在 Kb 序列中反查）。两个方向均采用线性插值。

### 7.1 第一步：选取两个最近的校准马赫数

从所有已加载的校准数据中，找出与当前马赫数距离最近的两个校准马赫数：

- calib1：距离最近的校准数据
- calib2：距离次近的校准数据

若只有一个校准文件，则 calib1 和 calib2 相同。

> **注意**：当前做法是按距离排序取最近两个，而非严格取上下界包围。这意味着当校准马赫数为 {0.2, 0.4, 0.6} 时，若当前 Ma=0.35，可能选到 (0.4, 0.2) 而非 (0.2, 0.4)。

### 7.2 第二步：在马赫数方向线性插值

对于攻角序列中的每一个角度位置，在两个校准马赫数之间做线性插值，得到当前马赫数下的系数。

**插值比例**：

$$
ratio = \frac{Ma_{current} - CMa_1}{CMa_2 - CMa_1}
$$

ratio 被限制在 [0, 1] 范围内。若两个校准马赫数相同（差值小于 $10^{-6}$），则 ratio = 0。

**对每个角度位置 i 的三个系数分别插值**：

$$
Kb_i = Kb_{1,i} + ratio \times (Kb_{2,i} - Kb_{1,i})
$$

$$
Kt_i = Kt_{1,i} + ratio \times (Kt_{2,i} - Kt_{1,i})
$$

$$
Sb_i = Sb_{1,i} + ratio \times (Sb_{2,i} - Sb_{1,i})
$$

其中下标 1、2 分别对应两个校准马赫数下的数据，i 对应同一个攻角位置。

**插值完成后**，得到一组新的映射表，每个条目包含当前马赫数下的 (Kb, Alpha, Kt, Sb)。

### 7.3 第三步：按 Kb 值排序

将上一步得到的映射表按 Kb 值从小到大排序。

> **前提假设**：Kb 随攻角单调变化（通常三孔探针在设计攻角范围内满足此条件）。若 Kb 不单调，排序后攻角与 Kb 的对应关系可能出错。

### 7.4 第四步：在 Kb 方向插值反查攻角和系数

根据测量的 Kb_temp 值，在排序后的映射表中查找对应的攻角和系数。

**边界处理**：
- 若 Kb_temp 小于等于表中最小的 Kb → 直接返回最小 Kb 对应的数据点
- 若 Kb_temp 大于等于表中最大的 Kb → 直接返回最大 Kb 对应的数据点

**正常插值**：

线性搜索映射表，找到 Kb_temp 落在哪两个相邻数据点之间：

$$
Kb_j \leq Kb_{temp} \leq Kb_{j+1}
$$

然后在两点之间线性插值：

$$
r = \frac{Kb_{temp} - Kb_j}{Kb_{j+1} - Kb_j}
$$

$$
Alpha = Alpha_j + r \times (Alpha_{j+1} - Alpha_j)
$$

$$
Kt = Kt_j + r \times (Kt_{j+1} - Kt_j)
$$

$$
Sb = Sb_j + r \times (Sb_{j+1} - Sb_j)
$$

返回插值得到的 Alpha、Kt、Sb。

### 7.5 二维插值完整伪代码

```
输入：Kb_temp（测量的压力系数比）、Ma_current（当前马赫数）、所有校准数据

// 第一步：选最近的两个校准马赫数
按 |CMa - Ma_current| 升序排列所有校准数据
calib1 ← 排名第1的校准数据
calib2 ← 排名第2的校准数据（若只有一个则与 calib1 相同）

// 第二步：马赫数方向插值，构建当前马赫数下的系数表
ratio ← (Ma_current − calib1.CMa) / (calib2.CMa − calib1.CMa)，限制在 [0, 1]

新建空表 kbAlphaMap
对每个攻角位置 i（从 0 到 攻角总数-1）：
    Kb_i ← calib1.Kb[i] + ratio × (calib2.Kb[i] − calib1.Kb[i])
    Kt_i ← calib1.Kt[i] + ratio × (calib2.Kt[i] − calib1.Kt[i])
    Sb_i ← calib1.Sb[i] + ratio × (calib2.Sb[i] − calib1.Sb[i])
    将 (Kb_i, Alpha_i, Kt_i, Sb_i) 加入 kbAlphaMap

// 第三步：按 Kb 升序排序
kbAlphaMap 按 Kb 值升序排列

// 第四步：在 Kb 方向反查
若 Kb_temp ≤ kbAlphaMap 首项的 Kb：
    返回首项的 (Alpha, Kt, Sb)

若 Kb_temp ≥ kbAlphaMap 末项的 Kb：
    返回末项的 (Alpha, Kt, Sb)

否则，线性搜索找到相邻两点 j 和 j+1 使得 Kb_j ≤ Kb_temp ≤ Kb_{j+1}：
    r ← (Kb_temp − Kb_j) / (Kb_{j+1} − Kb_j)
    Alpha ← Alpha_j + r × (Alpha_{j+1} − Alpha_j)
    Kt ← Kt_j + r × (Kt_{j+1} − Kt_j)
    Sb ← Sb_j + r × (Sb_{j+1} − Sb_j)
    返回 (Alpha, Kt, Sb)
```

## 8. 马赫数计算公式推导

### 8.1 物理基础

等熵流动中，总压与静压之比与马赫数的关系为：

$$
\frac{P_t}{P_s} = \left(1 + \frac{\gamma - 1}{2} Ma^2 \right)^{\frac{\gamma}{\gamma - 1}}
$$

其中 $\gamma$ 为比热比，空气取 1.4。

### 8.2 反解马赫数

将上式反解得到马赫数：

$$
Ma = \sqrt{ \frac{2}{\gamma - 1} \left[ \left( \frac{P_t}{P_s} \right)^{\frac{\gamma - 1}{\gamma}} - 1 \right] }
$$

### 8.3 代入空气参数

$$
\frac{\gamma - 1}{\gamma} = \frac{0.4}{1.4} \approx 0.2857
$$

$$
\frac{2}{\gamma - 1} = \frac{2}{0.4} = 5
$$

因此最终公式为：

$$
Ma = \sqrt{ 5 \times \left[ \left( \frac{P_t}{P_s} \right)^{0.2857} - 1 \right] }
$$

### 8.4 实际计算

注意压力需要使用绝对压力（表压 + 大气压）：

$$
P_{t,abs} = Pt + Pa
$$

$$
P_{s,abs} = Ps + Pa
$$

$$
Ma = \sqrt{5 \times \left| \left( \frac{P_{t,abs}}{P_{s,abs}} \right)^{0.2857} - 1 \right|}
$$

> 公式中使用了绝对值，在正常物理情况下 $P_{t,abs} > P_{s,abs}$，内部表达式应为正值。绝对值是一种防御性处理。

**异常处理**：若 $P_{s,abs} < 10^{-6}$（接近零），直接返回 Ma = 0。

---

## 9. 步骤3：最终计算输出

迭代收敛后，用最终的马赫数再做一次二维插值，得到最终的系数，然后计算所有输出量：

```
（Alpha_final, Kt_final, Sb_final）← 二维插值查找（Kb_temp, Ma_final）

PtProbe   ← P2 + Kt_final × ΔP
PsProbe   ← PtProbe − Sb_final × ΔP
MachProbe ← 等熵公式反算（PtProbe, PsProbe, Pa）
AlphaProbe ← Alpha_final
IterationCount ← 实际迭代次数
```

**异常回退**：若二维插值无法返回有效结果（如校准数据不足），则：
- PtProbe = P2
- PsProbe = P2
- MachProbe = 当前马赫数
- AlphaProbe = 0

## 10. 完整公式汇总

### 10.1 中间量

| 符号 | 公式 | 说明 |
|------|------|------|
| ΔP | $2P_2 - P_1 - P_3$ | 压力差分 |
| Kb_temp | $\frac{P_3 - P_1}{\Delta P}$ | 测量的压力系数比 |

### 10.2 插值公式

| 方向 | 公式 | 说明 |
|------|------|------|
| 马赫方向 | $ratio = \frac{Ma - CMa_1}{CMa_2 - CMa_1}$，限制在 [0, 1] | 插值比例 |
| 马赫方向 | $Kb_i = Kb_{1,i} + ratio \times (Kb_{2,i} - Kb_{1,i})$ | Kb 插值 |
| 马赫方向 | $Kt_i = Kt_{1,i} + ratio \times (Kt_{2,i} - Kt_{1,i})$ | Kt 插值 |
| 马赫方向 | $Sb_i = Sb_{1,i} + ratio \times (Sb_{2,i} - Sb_{1,i})$ | Sb 插值 |
| 攻角方向 | $r = \frac{Kb_{temp} - Kb_j}{Kb_{j+1} - Kb_j}$ | 反查比例 |

| 攻角方向 | $Kt = Kt_j + r \times (Kt_{j+1} - Kt_j)$ | Kt 反查 |
| 攻角方向 | $Sb = Sb_j + r \times (Sb_{j+1} - Sb_j)$ | Sb 反查 |

### 10.3 压力和马赫数计算

| 符号 | 公式 | 说明 |
|------|------|------|
| Pt | $P_2 + Kt \times \Delta P$ | 总压（表压） |
| Ps | $Pt - Sb \times \Delta P$ | 静压（表压） |
| Ma | $\sqrt{5 \times \left\| \left(\frac{Pt + Pa}{Ps + Pa}\right)^{0.2857} - 1 \right\|}$ | 马赫数 |

## 12. 算法特点与局限

1. **二维线性插值**：马赫数和攻角两个方向均采用线性插值，适用于校准数据密度较高的情况
2. **迭代求解**：马赫数通过反复修正逐步收敛，最多 20 次，收敛条件为马赫数变化量 < $10^{-4}$
3. **Kb 单调性前提**：算法假设 Kb 随攻角单调变化，这样才能通过 Kb 反查攻角。若超出有效攻角范围，Kb 可能不再单调
4. **最近邻选择**：选取校准马赫数时使用距离排序而非严格的上下界包围，在校准马赫数切换边界可能产生不连续
5. **绝对值防御**：马赫数公式中使用了绝对值，虽然增强了数值稳定性，但也可能掩盖静压大于总压的非物理情况
6. **预留的二次拟合**：数据结构中已定义了二次多项式拟合系数（$Kb = a \cdot Ma^2 + b \cdot Ma + c$），但当前算法未启用，仍使用线性插值
